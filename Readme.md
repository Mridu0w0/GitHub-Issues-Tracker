1️⃣ What is the difference between var, let, and const?
Ans: var is function-scoped, let is block-scoped and reassignable, and const is block-scoped but cannot be reassigned.

2️⃣ What is the spread operator (...)?
Ans:It unpacks like an array or object into individual elements or properties.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans: map() returns a new transformed array, filter() returns a new array of items that pass a condition, and forEach() simply runs a function on each item without returning a new array.

4️⃣ What is an arrow function?
Ans:It is a concise syntax for writing function expressions (() => {}) that does not bind its own this context.

5️⃣ What are template literals?
Ans:They use backticks (`) to easily create multi-line strings and embed variables directly into the text using ${}.
